# Lernziele - RechenStar

## Überblick

RechenStar orientiert sich an den Bildungsstandards für Mathematik in der Primarstufe (Klasse 1) und fördert grundlegende arithmetische Kompetenzen.

## Hauptlernziele

### 1. Zahlenverständnis (Zahlenraum 1-10)

#### Kompetenzen:
- Zahlen sicher erkennen und benennen
- Mengen zu Zahlen zuordnen
- Zahlenreihe vorwärts und rückwärts
- Vorgänger und Nachfolger bestimmen

#### In RechenStar:
- Visuelle Zahlendarstellung
- Mengenbilder bei Bedarf
- Zahlenstrahl-Visualisierung
- Finger-Counting erlaubt

### 2. Addition im Zahlenraum 1-10

#### Kompetenzen:
- Plusaufgaben verstehen
- Ergebnisse bis 10 sicher berechnen
- Verschiedene Rechenstrategien nutzen
- Tauschaufgaben erkennen (3+2 = 2+3)

#### Progression:
```
Stufe 1: Ergebnis ≤ 5
- 1+1, 2+1, 1+2, 2+2, 3+1, 1+3

Stufe 2: Ergebnis ≤ 7
- 3+2, 2+3, 4+1, 1+4, 3+3

Stufe 3: Ergebnis ≤ 10
- 5+3, 4+4, 6+2, 7+1, 5+5
```

### 3. Subtraktion im Zahlenraum 1-10

#### Kompetenzen:
- Minusaufgaben verstehen
- Ergebnisse ≥ 0 berechnen
- "Wegnehmen" als Konzept
- Umkehrung der Addition verstehen

#### Progression:
```
Stufe 1: Kleine Differenzen
- 3-1, 4-1, 5-2, 4-2

Stufe 2: Mittlere Differenzen
- 7-3, 8-2, 9-4, 6-3

Stufe 3: Größere Differenzen
- 10-5, 10-7, 9-6, 8-5
```

## Teilkompetenzen

### A. Rechenstrategien

#### Zählstrategien:
- **Weiterzählen**: 5+2 → 5...6,7
- **Rückwärtszählen**: 8-3 → 8...7,6,5
- **Alleszählen**: Neuanfang bei 1

#### Ableitungsstrategien:
- **Verdoppeln**: 3+3, 4+4, 5+5
- **Fast-Verdoppeln**: 3+4 (3+3+1)
- **Zehnerzerlegung**: Vorbereitung

### B. Mathematische Prozesse

#### Problemlösen:
- Aufgabe verstehen
- Lösungsweg finden
- Ergebnis überprüfen
- Verschiedene Wege kennen

#### Kommunizieren:
- Rechenwege erklären (visuell)
- Fehler verstehen
- Hilfe annehmen

#### Modellieren:
- Bilder zu Aufgaben
- Aufgaben zu Bildern
- Alltagsbezüge herstellen

## Lernpfad-Design

### Phase 1: Orientierung (Woche 1-2)
```
Ziel: Zahlenraum erkunden
- Zahlen 1-5 sicher
- Erste Plusaufgaben
- Mengenverständnis
```

### Phase 2: Grundlagen (Woche 3-6)
```
Ziel: Basis-Operationen
- Plus bis 10
- Minus einfach
- Automatisierung beginnen
```

### Phase 3: Festigung (Woche 7-10)
```
Ziel: Sicherheit gewinnen
- Alle Aufgabentypen
- Schnelligkeit steigern
- Strategien entwickeln
```

### Phase 4: Vertiefung (Woche 11-12)
```
Ziel: Flexibilität
- Gemischte Aufgaben
- Transfer-Aufgaben
- Knobelaufgaben
```

## Differenzierung

### Für leistungsstarke Kinder:
- Zahlenraum bis 20 (optional)
- Kettenaufgaben (2+3+1)
- Zeit-Challenges
- Knobelecke

### Für Kinder mit Förderbedarf:
- Zahlenraum 1-5 länger
- Mehr visuelle Hilfen
- Kleinere Schritte
- Mehr Wiederholungen

### Für Kinder mit Lernschwierigkeiten:
- Alternative Darstellungen
- Multisensorisch
- Individuelles Tempo
- Erfolgserlebnisse garantiert
