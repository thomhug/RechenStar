1. **Autonomie**: Wahlmöglichkeiten, Selbstbestimmung

## Psychologische Prinzipien

### Verstärkungslernen:
```
Positiv:
✓ Variable Verstärkung (manchmal Bonus)
✓ Sofortige Rückmeldung
✓ Soziale Verstärkung (Lob)

Negativ (vermeiden):
✗ Bestrafung für Fehler
✗ Zeitdruck
✗ Öffentliche Vergleiche
```

### Flow-Design:
- **Klare Ziele**: Kind weiss, was zu tun ist
- **Sofortiges Feedback**: < 1 Sekunde
- **Balance**: Herausforderung = Fähigkeit
- **Konzentration**: Keine Ablenkungen
- **Kontrolle**: Kind bestimmt Tempo

### Selbstwirksamkeit fördern:
1. **Mastery Experiences**: Erfolgserlebnisse
2. **Vicarious Learning**: Charaktere zeigen "du kannst das"
3. **Verbal Persuasion**: Ermutigung
4. **Emotionale Zustände**: Angstfrei lernen

## Entwicklungspsychologische Aspekte

### Kognitive Entwicklung (7 Jahre):
- **Arbeitsgedächtnis**: 3-4 Items
- **Aufmerksamkeit**: 10-15 Min fokussiert
- **Abstraktion**: Beginnend, noch konkret
- **Metakognition**: Entwickelt sich

### Emotionale Entwicklung:
- **Emotionsregulation**: Noch lernend
- **Frustrationstoleranz**: Begrenzt
- **Selbstkonzept**: Formend
- **Leistungsmotivation**: Intrinsisch dominant

### Soziale Entwicklung:
- **Peer-Vergleiche**: Werden wichtiger
- **Kooperation**: Bevorzugt vor Wettbewerb
- **Anerkennung**: Wichtig von Autoritäten

## Gamification-Strategien

### Gesunde Mechaniken:
✓ **Progress Bars**: Sichtbarer Fortschritt
✓ **Achievements**: Für Anstrengung, nicht nur Ergebnis
✓ **Narrativ**: Geschichte gibt Kontext
✓ **Customization**: Persönlicher Avatar
✓ **Feedback**: Informativ, nicht bewertend

### Zu vermeidende Dark Patterns:
✗ **FOMO**: "Nur heute verfügbar!"
✗ **Sunk Cost**: "Du verlierst alles!"
✗ **Social Pressure**: Rankings/Vergleiche
✗ **Variable Rewards**: Zu stark (Glücksspiel)
✗ **Endless Treadmill**: Kein Ende

## Verhaltenspsychologie

### Habit Loop:
```
1. CUE: App-Icon sichtbar / Eltern-Erinnerung
2. ROUTINE: 10 Minuten Mathe üben
3. REWARD: Sticker + Erfolgsgefühl
4. INVESTMENT: Sammlung wächst
```

### Motivation aufrechterhalten:
- **Novelty**: Neue Inhalte regelmässig
- **Mastery**: Gefühl von Verbesserung
- **Purpose**: "Ich lerne für die Schule"
- **Autonomy**: Eigene Entscheidungen

## Emotionales Design

### Positive Emotionen fördern:
- **Freude**: Durch Erfolge und Überraschungen
- **Stolz**: Leistung anerkennen
- **Neugier**: Was kommt als nächstes?
- **Zufriedenheit**: Session-Abschluss

### Negative Emotionen managen:
- **Frustration**: Sofort abfangen mit Hilfe
- **Angst**: Kein Zeitdruck, keine Strafen
- **Langeweile**: Variation und Anpassung
- **Scham**: Fehler normalisieren
